Quickly add a Remitano payment button anywhere using shortcode to get paid by cryptocurrency.

== Description ==

The Remitano Payment Button is a simple tool that allows users to quickly pay for goods and services using cryptocurrencies such as USDT, Bitcoin, and Ethereum using Remitano Payment Gateway. This payment method is adaptable to businesses of all sizes, particularly online stores. Besides, merchants have the opportunity to **expand their business to over 3 Million Remitano global cryptocurrency investors.**

**Remitano payment service:**

= Payment options =

Customers are flexible to choose one of the three available options:
* Use Remitano coin wallet
* Buy cryptocurrency directly within 5 minutes with fiat currencies such as: NGN, SGD, VND...
* Deposit coins into the wallet (zero confirmation)

= Supported coins =

Remitano currently supports payments in **6 cryptocurrencies**:

* Bitcoin (BTC)
* Ethereum (ETH)
* Tether (USDT)
* Ripple (XRP)
* Litecoin (LTC)
* Bitcoin Cash (BCH)

= Service charge =

* No transaction fee
* No registration fee
* No maintenance fee

Find out more about Remitano Payment Gateway [here](https://remitano.com/btc/payment_gateway) and Remitano Payment Button [here](https://developers.remitano.com/docs/payment-gateway/payment-buttons).
